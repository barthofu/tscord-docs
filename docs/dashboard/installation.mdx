# Installation

## Requirements

- **[Node.js](https://nodejs.org/)** (*>= 16.14.0*)
- **[NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)**

## Download the template

As for now, TSCord Dashboard is a **template** kind of github repository.

1. Click on the green button `Use this template` on the github repository
2. Clone the repo you've just created using command `git clone https://github.com/<you_username>/<your_repo>.git`

## Install the dependencies

Now, navigate to the cloned repo on your local machine and run the following command:
```bash
npm install
```
