# Configuration

:::caution
Before anything else, head over to your **bot** config and enable the API server.
:::

1. Fill the `.env.example` with the API URL and API key of your TSCord based bot
2. Rename it `.env`
3. Fill the `config.json` as you want
4. Make sure your bot is running and that the API is accessible
5. Run `npm run build`
