# Docker

TSCord is fully configured to work flawlessly with **[Docker](https://www.docker.com/)** using **[docker-compose](https://docs.docker.com/compose/)**.

Just install it and run the `docker-compose up -d` command to spin up a production ready instance of your bot!

:::tip
To install `docker-compose`, head over **[here](https://docs.docker.com/compose/install/)**.
:::
